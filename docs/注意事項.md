# 注意事項

## 1. 環境準備

本專案是一個基於 Python 的 Flask 應用程式，因此您需要先安裝 Python 環境。

1.  **安裝 Python**: 如果您的系統尚未安裝 Python，請從官方網站下載並安裝：[https://www.python.org/downloads/](https://www.python.org/downloads/)。建議在安裝過程中勾選「Add Python to PATH」選項。

## 2. 安裝必要的套件

在啟動應用程式之前，您需要安裝所有依賴的 Python 套件。

1.  開啟命令提示字元 (cmd) 或 PowerShell。
2.  導航到您的專案根目錄 (`C:\Users\Bruce\Desktop\天氣\`)。
3.  執行以下命令來安裝 `requirements.txt` 中列出的所有套件：
    ```bash
    python -m pip install -r requirements.txt
    ```

## 3. 啟動應用程式

完成套件安裝後，您可以啟動後端伺服器。

1.  在專案根目錄下，執行以下命令啟動 Flask 應用程式：
    ```bash
    python backend/app.py
    ```
    伺服器啟動後，您將會在終端機中看到類似「Running on http://127.0.0.1:5000」的訊息。

## 4. 訪問應用程式

伺服器成功啟動後，您可以在網頁瀏覽器中訪問以下網址來使用天氣應用程式：

[http://127.0.0.1:5000](http://127.0.0.1:5000)
