# 📊 PPT 簡報內容 (依照投影片分頁)

這是專門為 PowerPoint 設計的格式，您可以直接複製標題與內文到簡報中。

---

## [Slide 1] 前端功能：核心體驗
**標題：直觀的天氣查詢體驗**

*   **📍 自動定位**：GPS 自動偵測，秒速鎖定所在城市。
*   **🔍 關鍵字快搜**：支援中英文輸入 (如: 台中/Taipei)。
*   **🌥️ 即時數據**：溫度、體感、濕度、紫外線一目了然。
*   **📅 未來一週預報**：7 天溫差趨勢與降雨機率預測。

---

## [Slide 2] 前端功能：智慧加值
**標題：不只是天氣，更是生活助手**

*   **👗 智慧穿搭**：依據「體感溫度」推薦洋蔥式穿搭或帶傘。
*   **😷 空氣品質 (AQI)**：紅綠燈號警示，即時掌握 PM2.5 狀況。
*   **🗺️ 動態雷達圖**：整合氣象署回波圖，視覺化雨雲動向。
*   **🎤 語音互動**：支援「語音輸入查詢」與「天氣語音播報」。   

---

## [Slide 3] 後端技術：數據處理
**標題：強大的 Python 運算核心**

*   **📡 雙 API 串接**：整合氣象署 (CWA) 與 環境部 (EPA) 開放資料。
*   **⚙️ 資料解析與正規化**：解析複雜 JSON 結構，將原始資料轉換為統一格式。
*   **💾 持久化儲存**：SQLite 資料庫自動記錄每一次查詢。
*   **📊 大數據分析**：Pandas 計算平均溫、歷史極值與趨勢。

---

## [Slide 4] 後端技術：進階架構
**標題：專業級系統設計**

*   **👮 背景災害監控**：Threading 多執行緒，24小時監控颱風警報。
*   **🛡️ 系統穩定性防護機制**：導入 Try-Except 容錯設計，防止 API 連線異常造成當機。
*   **📦 資料報表匯出**：一鍵將資料庫轉為 Excel 報表下載。
*   **🧱 模組化開發**：MVC 架構分離 (API, Logic, View)，易於維護。
